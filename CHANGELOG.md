# Changelog

All notable changes to Phantom are documented in this file.

## 2.2.3 - 2026-02-21

- Metadata hardening: moved sender attribution into encrypted message envelope (`PHANTOM_MSG_V1`)
- Metadata hardening: reduced file metadata leakage in upload flow
- Compatibility handling for legacy and envelope-based payloads
- Documentation cleanup and repo hygiene updates

## 2.2.2 - 2026-02-21

- Added custom local MOTD profile support (no URL fetch required)
- Added settings flow to edit and clear custom MOTD lines
